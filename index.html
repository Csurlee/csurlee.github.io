DNS & DHCP

Diagnostics

Full Coverage

Every corner of pfSense,
in one app

WallWatch wraps the entire pfSense REST API v2 surface — no feature left behind.

  • 📡
    Routing & Gateways
    Static routes, gateway groups, priority policies, dynamic routing awareness.
  • 🔐
    Authentication & Users
    Local users, API key management, groups, privileges — full RBAC.
  • 📦
    NAT Rules
    Outbound NAT, port forwards, 1:1 NAT with full create/edit/delete.
  • 🗄
    Aliases
    IP, port, URL and nested alias management with type-specific entry editors.
  • 📊
    Status & Monitoring
    System dashboard, gateway status, service status with start/stop/restart actions.
